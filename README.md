This is not officially released as the workflow is not complete and it's a work in progress, but it should be able to build via MSBuild with the provided solution (`.sln`) file. This should be built in *`C++17`* and warning level 4 (`/W4`) is recommended. For now the few functions provided can be viewed in the header source [Header.cpp](./AutoHotkeyFlux/AutoHotkeyFlux/Header.cpp) (temporary generic name). This is somewhat hackish code (to make it work well with AHK), but it's been checked for memory leaks and safety functions are used where possible.
